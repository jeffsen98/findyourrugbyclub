To set up rankings:

  1. Open your Google Sheet and add a new tab named Rankings
  2. Use these headers in row 1: rank, team, division, conference, w, l, d
  3. Fill in your data — set division to D1, D2, or D3 for each row
  4. File → Share → Publish to web → select the Rankings tab → CSV → Publish
  5. Paste the URL into rankings.html at https://docs.google.com/spreadsheets/d/e/2PACX-1vSUu5J74G1QqUbH0fDgKBvYj-4CWs51yc0LRaPf-TFTLs9hUrM-h1p6BzcSLKeFSxwtyTG9HpK63y_J/pub?gid=1696848951&single=true&output=csv
